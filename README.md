# Sports Management System

This is a comprehensive sports management system with both frontend and backend components.

## Project Structure

### Backend
- Node.js/Express server
- MongoDB database
- RESTful API endpoints
- Authentication system
- Models for Users, Events, Courts, and Memberships

### Frontend
- React-based UI
- Material-UI components
- Responsive design
- Multiple pages for different features

## Setup Instructions

### Backend Setup
1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Create a .env file with your configuration
4. Start the server:
```bash
npm start
```

### Frontend Setup
1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
